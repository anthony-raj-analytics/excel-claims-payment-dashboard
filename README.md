# Excel Claims Payment Dashboard

An Excel-based analytical dashboard designed to analyze healthcare claims payment trends and performance using cleaned data.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Power Query
- Charts & Slicers

## Data Source
- Cleaned dataset used for analysis
- Raw data not available
- Data organized across multiple sheets within a single Excel workbook

## Project Files

- **excel/claims_analysis_dashboard.xlsx**  
  This Excel workbook contains:
  - **Clean_Data** sheet – cleaned and structured dataset
  - **Pivot_Analysis** sheet – pivot tables and calculations
  - **Dashboard** sheet – final interactive Excel dashboard

- **images/excel-dashboard.png**  
  Dashboard preview image

## Use Case
Healthcare claims payment analysis to understand payment trends, category-wise distribution, and processing delays.

## Key Insights
- Identification of the highest claim payment categories
- Analysis of delays in claims payment processing
- Monthly and category-wise payment trend analysis

## Assumptions
- Dataset is pre-cleaned and validated
- No missing or duplicate records in the provided data

## Outcome
This dashboard helps stakeholders quickly identify payment patterns, delays, and high-impact claim categories, enabling better operational and financial decision-making.
